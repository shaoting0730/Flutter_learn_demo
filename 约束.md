# 遵循向下传递约束，向上传递尺寸的条件
Tight Constraint: 紧约束，最小约束等于最大约束。 <br/>
Loose Constraint: 松约束，约束最小等于0.  <br/>
ps:如果一个约束，其约束最小值为0，约束最大值也为0，则该约束既是紧约束又是松约束。 <br/>
